# 📦 Smart Inventory API

**Smart Inventory** — это учебный pet-проект для системы учёта вещей. 
Создан в рамках производственной практики 3-го курса.

## 🚀 Основные функции
*   ✅ Добавление предметов в базу данных (POST)
*   ✅ Просмотр полного списка инвентаря (GET)
*   ✅ Удаление предметов по ID (DELETE)
*   ✅ Автоматическая документация (Swagger)

## 🛠 Стек технологий
*   **Язык:** Python 3.10+
*   **Фреймворк:** FastAPI
*   **БД:** PostgreSQL + SQLAlchemy
*   **Контейнеризация:** Docker & Docker-compose
