# 📦 Smart Inventory API

**Smart Inventory** — это учебный проект для системы учёта вещей. 

## 🚀 Основные функции
*   ✅ Добавление предметов в базу данных (POST)
*   ✅ Просмотр полного списка инвентаря (GET)
*   ✅ Удаление предметов по ID (DELETE)
*   ✅ Автоматическая документация (Swagger)

## 🛠 Стек технологий
*   **Язык:** Python 3.10+
*   **Фреймворк:** FastAPI
*   **БД:** PostgreSQL + SQLAlchemy
*   **Контейнеризация:** Docker & Docker-compose

## ⚙️ Инструкция по запуску
Для запуска проекта вам понадобится установленный **Docker**.

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Duke13337/praktika.git
2. Запустите проект (сборка контейнеров):
   ```bash
   docker-compose up --build
3.Приложение будет доступно по адресу:
🔗 http://localhost:8000/docs
